##Description

The software allows for one to read an excel file in order to loaf its data, while sorting it depending on the given attribute. In addition it creates a polynomial function that represents the pattern of data with a certain attribute. Finnaly it can graph a histogram, again depending on the given attribute.

##Dependencies

Check that the latest version of Python is installed, if not install it through pythons website https://www.python.org/.

Install Matplotlib and Numpy libraries

Installing numpy and matplotlib in the command propt terminal:
C:> python -m pip install matplotlib
C:> python -m pip install numpy


##Installation

An integrated develpoment environment (IDE) that support python is required.
User should have a .csv/.txt file with data that can be processed further.

1. Unzip the dowenloaded file by extracting it
2. Open batcht_UI.py in the IDE if you wish to process a file based on a user interface.
3. Oen text_UI.py in the IDE if you wish to provide the inputs to the user interface.
4. Run the program in pythons' IDE, you should see the user interface and will be prompted to either enter a file or the inputs of your commands manually.
